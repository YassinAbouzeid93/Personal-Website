Privacy Policy - YDA Labs

Last updated: January 2025

YDA Labs (“we”, “our”, “us”) provides analytics and reporting services for advertisers. This Privacy Policy explains how we collect, use, and store data when advertisers authorize our app through the TikTok Marketing API.

1. Information We Access

We only access aggregated advertising performance data through the TikTok Ads API, including:

  • Ad accounts, campaigns, ad groups, and ads
  • Metrics such as impressions, clicks, conversions, CTR, CPC, CPM, ROAS, and spend

We do NOT access:

  • Personal user information
  • TikTok account login data
  • Messages, followers, or profile data

2. How We Use the Data

Data is used solely to provide:

  • Automated performance reporting
  • Optimization dashboards
  • Business insights for the advertiser

We do not sell, share, or use data for any other purpose.

3. Data Storage

Data is securely stored using encrypted systems such as Google Cloud. Access is restricted to authorized YDA Labs systems only.

4. Revoking Access

Advertisers may revoke API access anytime via: TikTok Ads Manager → Assets → Apps → Remove Access.

5. Contact

If you have questions about this policy, contact us:
